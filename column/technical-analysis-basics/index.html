テクニカル分析とは

テクニカル分析とは、過去の株価・出来高のデータをグラフ（チャート）で分析し、将来の株価の方向性を予測する手法です。「市場の価格はすべての情報を織り込んでいる」という前提のもと、チャートパターンや数学的指標から売買タイミングを判断します。

ファンダメンタルズ分析（企業の業績・財務分析）が「何を買うか」を決めるのに対し、テクニカル分析は「いつ買う・いつ売るか」のタイミングを決めるのに適しています。多くの投資家がテクニカル分析を組み合わせて使っています。

ローソク足チャートの読み方

ローソク足は1日（または週・月）の株価の動きを「始値・高値・安値・終値」の4本値で表示します。終値が始値より高い（上昇した日）は白・緑の「陽線」、終値が始値より低い（下落した日）は黒・赤の「陰線」で表示されます。

長い上ひげ（上に長いひげ）は「高値圏での売り圧力が強い」サイン、長い下ひげは「安値圏での買い意欲が強い」サインとして解釈されます。

移動平均線（MA）

移動平均線は直近N日間の終値の平均を結んだ線です。よく使われるのは25日線（約1ヶ月）・75日線（約3ヶ月）・200日線（約1年）です。

ゴールデンクロス：短期線（25日）が長期線（75日）を下から上に突き抜けるパターン。上昇トレンドへの転換シグナルとして有名です。

デッドクロス：短期線が長期線を上から下に突き抜けるパターン。下落トレンドへの転換シグナルです。

移動平均線より株価が上にある状態（株価＞移動平均線）はトレンドが強い、下にある状態は弱いと判断します。

RSI（相対力指数）

RSIは0〜100の数値で表される「買われすぎ・売られすぎ」を示す指標です。一般的に70以上で「買われすぎ（過熱）」、30以下で「売られすぎ（底値圏）」と判断されます。

RSIが70を超えて高値をつけた後に70を割り込んだ場合は利益確定のサイン、30を下回った後に30を上回った場合は底打ちサインとして使われます。

ただしトレンドが強い上昇相場ではRSIが70以上でも上昇が継続することが多く、RSI単独での判断は避け、移動平均線や出来高と組み合わせることが重要です。

MACD（移動平均収束拡散法）

MACDは短期EMA（指数平滑移動平均）と長期EMAの差（MACDライン）と、そのシグナルライン（MACDの移動平均）を組み合わせた指標です。

MACDラインがシグナルラインを下から上に突き抜ける「ゴールデンクロス」が買いシグナル、上から下に突き抜ける「デッドクロス」が売りシグナルです。MACDがゼロラインを上回っているときは上昇トレンド、下回っているときは下降トレンドを意味します。

テクニカル分析の注意点

テクニカル分析は「過去のパターンの繰り返し」を前提にしていますが、未来は必ずしも過去と同じではありません。決算発表・政策変更・地政学的イベントなどのファンダメンタルな要因により、テクニカルシグナルが無効化されることがあります。

複数の指標が同時にシグナルを発したとき（例：移動平均ゴールデンクロス＋RSI底打ち＋出来高急増）は信頼度が高くなります。テクニカル分析はシグナルの重ね合わせ（コンフルエンス）が重要です。
