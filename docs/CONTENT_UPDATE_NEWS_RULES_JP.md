# StockWaveJP コンテンツ更新時のお知らせ運用

以下を更新した場合は、必ず同じコミットでお知らせも更新します。

対象:
- 週次・月次・四半期レポート
- コラム・解説
- 公式テーマの追加・削除
- テーマ構成銘柄の追加・削除・入れ替え
- 料金、プラン、更新頻度
- 主要機能の追加・変更・障害修正

更新箇所:
1. frontend/src/components/pages/News.jsx
   - MANUAL_NEWS の先頭に詳細なお知らせを追加
2. frontend/src/components/pages/TopPage.jsx
   - ALL_NEWS の先頭にホーム用の短いお知らせを追加

記載ルール:
- 日付は YYYY/MM/DD
- 何を変更したか
- 対象テーマ・銘柄・レポート期間
- 利用者への影響
- 数字が未取得なら推測値を書かない
