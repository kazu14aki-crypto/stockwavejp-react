# 株価指数・商標・分類データ利用レビュー

作成日: 2026-07-20

## 結論

現状のまま「問題ない」とは判断できません。

指数名を市場解説のために文章中で正しく参照することと、
指数値・構成銘柄・公式分類を商用サイトで継続表示することは別です。

StockWaveJPはサブスクリプション提供を予定する情報サービスなので、
日経指数は日本経済新聞社、TOPIX等はJPX総研へ利用形態を説明し、
書面でライセンス要否を確認してください。

## 現在のコードで確認した事項

### 日経平均関連

backend/data.py は、日経平均株価の構成銘柄を
「技術・金融・消費・素材・資本財その他・運輸公共」の6分類へ整理し、
市場別詳細用に `日経225｜分類名` のセグメントを作っています。

この6分類と構成銘柄の組合せは日経平均プロフィルの公式分類と近く、
一般的な独自分類とは言いにくい状態です。

日経の公式案内では、指数構成銘柄リストを対外表示する場合、
日経のデータサービス等から取得し、データ利用契約が必要とされています。

### TOPIX関連

backend/data.py にはTOPIX連動型ETFのほか、`^TOPIX` を
「TOPIX指数」として扱う設定が残っています。

ETF価格を使うことだけで、商標・指数データ・価格データの
ライセンス問題が自動的に解消されるわけではありません。
ETF価格のサイト表示可否は価格データ提供契約、
指数名や指数値の利用は指数権利者の条件を別々に確認します。

市場平均の代理値に1306を使う場合は、画面上で
「TOPIX連動型上場投信（1306）の騰落率を市場代理値として使用」
と明記し、「TOPIX指数値」と誤認させない表示にしてください。

### TOPIXの業種分類について

証券コード協議会の33業種分類と、JPX総研が算出する
東証業種別株価指数・TOPIX-17は区別が必要です。

分類名を説明目的で参照する場合でも、公式の銘柄別分類データを
一括取得・再配布する場合は、取得元の利用条件を確認してください。

公式指数と同じ構成銘柄、同じ分類、同じ計算方法で数値を作り、
TOPIX業種指数に見える名称で公開することは避けてください。

## 推奨する対応

### 最も安全

1. 日本経済新聞社へ次を問い合わせる
   - 日経平均の名称を市場解説やETF名で表示
   - 日経平均の構成銘柄を一覧表示
   - 公式6分類を使って独自の単純平均を算出
   - 無料・有料会員へウェブ配信

2. JPX総研へ次を問い合わせる
   - TOPIX名称、指数値、騰落率を表示
   - TOPIX連動ETFを市場代理値として利用
   - 33業種またはTOPIX-17の分類・構成銘柄を表示
   - 分類別の独自騰落率を算出して配信

3. Infowayへ次を書面確認する
   - 指数値の提供元と再配布権
   - ETF価格の無料・有料会員向け表示
   - 銘柄の市場区分・業種コードの再配布
   - 派生値、単純平均、市場超過値の公開

### ライセンス確認までの暫定対応

- 日経平均・TOPIXの「指数値」は表示しない
- 市場代理値はETF騰落率であることを明記
- 日経平均公式6分類と構成銘柄を使う「国内主要株」を一時非公開にする
- 独自分類へ切り替える場合は
  「StockWaveJP独自分類」「単純平均」「公式指数ではない」と明記
- 33業種を使う場合は、契約済みデータの業種コードを使用し、
  「証券コード協議会の33業種分類に準拠」と出典を記載

## 推奨表示例

市場代理値:
「市場比較にはTOPIX連動型上場投信（1306）の騰落率を使用しています。
TOPIX指数そのものの値ではありません。」

独自分類:
「本分類および騰落率はStockWaveJPが独自に構成・算出した参考値です。
日本経済新聞社、JPX総研その他の指数提供者が算出・公表する
公式指数ではなく、各社の推奨・承認を受けたものではありません。」
