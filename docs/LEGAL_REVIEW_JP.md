# StockWaveJP 法務文書レビュー記録（2026-07-22）

対象:
- 利用規約
- プライバシーポリシー
- 免責事項

主な修正:
- 14日間無料体験・自動課金なしへ統一
- 解約・プラン変更をプラン・料金ページとStripe Customer Portalへ統一
- アカウント削除と有料契約解約が別処理となる可能性を明記
- 過度な全面免責を避け、故意・重過失・強行法規の例外を明記
- ランキング、注目、シグナルが投資推奨ではないことを明記
- 未取得、更新失敗、ゼロ、推定値、市場代理値、独自分類を区別
- Googleログイン、Supabase、Stripe等で取得する情報を具体化
- 実装未確認のGDPR・CCPA・広告利用の断定を削除
- 保存期間、国外処理、安全管理、削除後に残る法定記録を明記

正式リリース前の必須確認:
- 特定商取引法に基づく表示の氏名、住所、電話番号、価格、支払時期、提供時期、解約条件
- Stripe最終確認画面の契約期間、自動更新、総額、解約条件
- 実際の料金、無料体験、更新頻度と全ページの一致
- 外部サービスの実際の利用状況
- データ提供会社の商用表示、再配布、派生値公開の許諾
- 個別の売買タイミングや投資額を利用者ごとに指示しない運用
- 弁護士による最終確認

本記録は開発上の論点整理であり、弁護士による法律意見ではありません。
