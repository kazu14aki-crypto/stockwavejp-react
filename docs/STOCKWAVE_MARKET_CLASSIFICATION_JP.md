# StockWaveJP独自市場分類

## 位置づけ

この分類は、日経平均株価の分類、TOPIXの業種分類、証券取引所の市場区分、その他の公式指数分類ではありません。
StockWaveJPが、日本株市場の大まかな資金移動を比較するために作成した編集上の独自分類です。

## 対象ユニバース

代表的な大型・中大型株150銘柄を対象に、各社の主力事業を基準として原則1社1分類で割り当てます。
時価総額指数を再現するものではなく、分類別騰落率は構成銘柄の単純平均です。

## 10分類

- デジタル・半導体: 24銘柄。半導体、電子部品、IT機器、ソフトウェアなど、デジタル化を支える企業群。
- 産業機械・防衛: 14銘柄。工場設備、産業機械、計測・制御、防衛・航空関連を中心とする企業群。
- 自動車・モビリティ: 18銘柄。自動車、部品、鉄道、海運、物流など、人と物の移動を担う企業群。
- 金融・商社: 19銘柄。銀行、証券、保険、投資サービス、総合商社をまとめた企業群。
- ヘルスケア: 14銘柄。医薬品、医療機器、診断・検査、ヘルスケア関連の企業群。
- 消費・小売・エンタメ: 8銘柄。小売、日用品、食品、ゲーム・娯楽など個人消費に近い企業群。
- 通信・ネットサービス: 7銘柄。通信、インターネット、求人・プラットフォーム型サービスの企業群。
- 素材・化学: 19銘柄。鉄鋼、非鉄、化学、繊維、ガラス、セメントなど産業の基礎素材を担う企業群。
- エネルギー・公益: 13銘柄。電力、ガス、石油、資源開発などエネルギー供給を担う企業群。
- 建設・不動産・住環境: 14銘柄。建設、住宅、不動産、住宅設備など都市・住環境を支える企業群。

## 運用ルール

- 公式指数や市場区分の構成銘柄一覧をそのまま複製しない
- 1社を原則1分類とし、分類間の重複による平均値の歪みを避ける
- 主力事業が変化した場合は分類を見直す
- 構成変更時はお知らせへ追加・削除・変更理由を記載する
- 株価が下落したことだけを理由に構成から除外しない
- 「公式指数ではない」「単純平均」であることを画面上に明示する
